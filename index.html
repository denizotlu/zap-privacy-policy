Privacy Policy

1. Data Collection and Usage

Our application does not collect any personal data from users. All data displayed within the app (standings, match results, statistics) is obtained from third-party APIs.

2. API Usage Limits

Our application is subject to free API usage limits:

  • Maximum 10 API requests per minute
  • Users will be notified in case of limit exceedance

3. Data Caching

For performance optimization:

  • League standings are cached for 3600 seconds
  • Prediction results are cached for 3600 seconds

4. User Data

  • No personal user data is stored
  • App preferences are stored locally on the device
  • Internet connection is required
